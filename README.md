# Sales & Shipping Analysis Dashboard
# Overview
This project focuses on analyzing sales performance, profit trends, and shipping costs across different categories, regions, and time periods. 
The goal is to identify key business insights and support data-driven decision-making through interactive dashboards.

# Tools & Technologies
- Power BI  
- Microsoft Excel  
- SQL (for data extraction and querying)  
# Key Features
## Sales & Profit Analysis
- Total Sales: **13M**
- Total Profit: **1.47M**
- Total Orders: **51K**
- Profit Margin: **100%**
- Yearly sales trend analysis (2011–2014)
- Category-wise profit comparison (Technology, Office Supplies, Furniture)
- Identification of top-performing products based on sales
## Shipping Cost Analysis
- Average Shipping Cost: **26.38**
- Total Orders Analyzed: **51K**
- Shipping cost trends over time
- Regional shipping cost comparison
- Category-wise shipping cost distribution
- Analysis by shipping modes (First Class, Second Class, Same Day, Standard Class)
# Key Insights
- **Technology category** generates the highest profit and sales among all categories  
- **Sales show consistent growth** from 2011 to 2014  
- Certain regions have significantly **higher shipping costs**, impacting overall profitability  
- **Standard Class shipping** is the most commonly used mode  
- High-value products contribute disproportionately to total revenue
